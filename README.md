# 简单定投

一个把指数基金定投纪律转成操作建议的本地小应用。

## 使用方式

直接用浏览器打开 `index.html` 即可使用，不需要安装依赖。
也可以通过 `python3 -m http.server 8080` 启动本地服务后访问 `http://localhost:8080`。

## 主要功能

- **资金总览**：录入可投资金额，持仓总金额由基金清单自动汇总。核心仓/卫星仓的目标金额和已持仓金额在侧边栏清晰展示。
- **智能建议**：根据 PE 百分位和预设的投资纪律，给出买入、卖出或持有的具体操作建议，包含建议金额。
- **确认执行与撤回**：当建议买入或卖出时，可点击"确认执行已操作"按钮，系统自动更新持仓金额并记录操作时间。支持多次"撤回上一步"，直到恢复到初始状态。
- **定投周期控制**：可设置定投周期（默认 7 天）。确认执行后，在周期内即使估值仍处于低位也不会重复发出买入建议，帮助用户严守纪律。
- **仓位比例与再平衡**：核心仓/卫星仓比例可自定义（默认 70/30），实时可视化环形图展示当前仓位分布，并在设定的再平衡月份提供调仓建议。
- **估值自动获取**：接入蛋卷基金公开 API，自动获取指数 PE 及 PE 百分位，支持手动修正。
- **本地存储**：数据保存在浏览器 localStorage 中，下次打开仍会保留。

## Android 开发指南

本项目是通过 [Capacitor](https://capacitorjs.com/) 将 Web 版本转换而来的原生 Android 工程。

### 1. 目录结构
- `/Web`: **Web 版本源码目录**。所有的 HTML、CSS、JS 修改都在这里进行。
- `/Android`: **安卓原生工程目录**。存放 Android Studio 项目源码。
- `capacitor.config.json`: Capacitor 配置文件。
- `package.json`: 包含快捷命令的配置文件。

### 2. 开发与运行
1. **环境准备**: 安装 Android Studio 和 Android SDK (API 30+)。
2. **打开项目**: 使用 Android Studio 打开根目录下的 `Android` 文件夹。
3. **运行**: 连接手机或模拟器，点击 **Run**。
4. **同步代码**: 修改 `/Web` 后，执行 `npm run sync` 将资源同步到安卓工程。
5. **打包**: 通过 **Build** -> **Build APK(s)** 生成安装包。

## 技术架构

### 1. 技术栈
本项目遵循轻量、无构建工具的设计理念：
*   **HTML5/CSS3**: 原生布局与语义化标签。
*   **JavaScript (ES6+)**: 原生 JS (Vanilla JS)，无框架依赖。
*   **数据存储**: 使用浏览器原生 `localStorage` 进行本地状态持久化。

### 2. 核心状态管理
应用将所有状态收敛在全局 `state` 对象中：
*   **资金**: `availableCash` (总资金池)。
*   **规则**: `rules` (包含核心/卫星比例、定投步数 `dcaBatches`、周期 `dcaCycleDays` 等)。
*   **基金**: `funds` 列表。
*   **历史**: `actionHistory` 撤销栈。

### 3. 智能分批与分配算法 (Rolling DCA)
应用采用“总资金池 + 动态分批”算法，解决资金变动时的投入节奏问题：
1.  **计算本期预算**: `本期定投总预算 = 当前可用总资金 / 建议定投步数`。
2.  **识别理想需求**: 计算所有符合买入条件基金的“理想买入缺口”。
3.  **最终额度确认**: `本期建议投入总额 = Math.min(本期定投总预算, 总理想缺口)`。
4.  **比例分配**: 按各个基金缺口占总缺口的比例进行资金切分。

## 说明

这个工具只用于执行你预先写好的纪律规则，不构成投资建议。真实交易前仍需要核对基金公告、交易成本、税费、流动性和个人现金流。

## 关于 API

当前版本已接入蛋卷基金的公开指数估值 API，通过 codetabs 代理解决跨域问题，自动刷新指数 PE 和 PE 百分位。个人账户数据（如持仓金额）由用户手动填写。
